O projeto

Filetador e extrusora de garrafa pet

Este projeto consiste na fabricação viabilizando a fácil produção de matéria-prima para suprir as impressoras 3D.
Onde temos como parte do projeto, um filetador e uma extrusora de filamentos.
O filetador tem como papel transformar o PET em filetes pequenos, para que sejam extrudados pela extrusora.
A extrusora, que é feira com um Hot End da própria impressora 3D tem o papel de aquecer o filete de garrafa PET ao tal ponto que seja transformado num fio na espessura correta para se realizar impressões 3D.
Enquanto esse filete está sendo derretido, se faz necessário um sistema de tração para a extrusão da forma correta do respectivo filamento. Tração essa que é feita por um motor controlado por PWM via página web.
Sendo o responsável pela tração, um motor DC que atua juntamente com um sistema de engrenagens que possibilita a rotação de um carretel que armazenará este filamento pronto para uso.
